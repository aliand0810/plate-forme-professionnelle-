📲 Installer Bureau Virtuel Pro

Pour installer l'application sur votre ordinateur avec une icône sur le bureau, lancez-la via un serveur local — Chrome pourra alors l'installer comme une vraie application.

1
Utilisez le lanceur fourni
Double-cliquez sur Démarrer Bureau Virtuel.bat dans le même dossier que le fichier HTML. Il démarre automatiquement un serveur local et ouvre Chrome.
2
Ou ouvrez manuellement via localhost
Dans Chrome, ouvrez :
http://localhost:8765/Bureau Virtuel Professionnel.html
3
Cliquez sur "Installer" dans Chrome
Une icône 📲 apparaîtra dans la barre des tâches du bureau. Cliquez dessus, puis confirmez l'installation — une icône apparaîtra sur votre bureau et dans le menu Démarrer.
☁
Bonus : Installer depuis GitHub Pages
Si vous avez configuré GitHub, déployez sur GitHub Pages pour accéder à l'app via HTTPS et l'installer sur n'importe quel appareil.
